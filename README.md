🚀 AI Dashboard Project

📌 Overview

This project is an AI-powered data dashboard built using Streamlit.
It provides real-time insights, filtering, and interactive analytics on student data.

---

✨ Features

- 📊 Interactive Dashboard
- 🔍 Smart Filters (City & Marks)
- 🤖 AI Chat Assistant (Powered by GPT)
- 📈 Key Insights (Average, Highest, Lowest Marks)
- 📉 Extra Insights (Median & Standard Deviation)
- 🧠 Smart Student Search
- 🎨 Clean UI with modern design

---

🛠 Tech Stack

- Python
- Streamlit
- Pandas
- OpenAI API

---

📊 Dashboard Functionalities

🔹 Key Insights

- Total Students
- Average Marks
- Highest & Lowest Marks

🔹 AI Chat

- Ask questions about data
- Get AI-powered responses

🔹 Filters

- Filter by City
- Filter by Minimum Marks

🔹 Smart Assistant

- Search student details
- Get instant insights

---

🚀 How to Run

1. Clone the repository:
   git clone https://github.com/anjala30/ai-dashboard-project.git

2. Install dependencies:
   pip install -r requirements.txt

3. Run the app:
   streamlit run app.py
